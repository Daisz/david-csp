Python Game Project

Rock-Paper-Scissors-Lizard-Spock (RPSLS)
For this project, I made a game where the player can play Rock-Paper-Scissors, as well as Rock-Paper-Scissors-Lizard-Spock

The code shown to the left is how I defined "stuff" which is otherwise known as the plays a player can choose from in Rock Paper Scissors Lizard Spock. It also shows how the players input is returned to the user, along with the choice of the computer

The code shown to the left is how i defined 'l', which is used to decide the winner of the game. The variable 'l' is ran through a segment of code which compares it to the computer's choice in order to decide a winner for the game.

The code shown above displays a few outcomes of Rock-Paper-Scissors-Lizard-Spock, and the code to the right shows the docstrings in use, which could guide the user if they need help playing the game.

In case the user continues to fail at inputting an acceptable variable, there is a set of strings which will be printed to hopefully guide the user as to how the game should be played.

While writing the code for this project, I learned how to properly define a random variable. Initially, I had been redefining the random variable within the elif statements, resulting in
the users choice being ran against a newly randomized computer choice every time the computer would check for a winner.
Instead, I defined the random choice as a variable, in order to keep it consistent throughout the code each turn.
